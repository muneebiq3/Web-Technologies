Lab Task 3 (I) - Job Application Form

Lab Task 3 (I) involves creating a job application form for COMSATS University, focusing on non-faculty roles. The form includes several important sections:

  • Header Section: Displays the COMSATS University logo and the form's title, creating a branded and professional appearance.
  • Form Content: Collects key information from applicants, such as the applied position, personal details, NIC number, department preferences, and campus locations. It also includes fields for contact information and academic qualifications.
  • Campus Selection: Features multiple checkbox options for selecting campus preferences, giving applicants flexibility in indicating where they would like to work.
  • Table for Academic Background: Contains fields for applicants to enter their degree details, including year of award, field, board/university, and grade or division.
The form's visual design, managed through external CSS (`style.css`), ensures clarity and a clean layout, with attention to usability and responsive behavior.

Lab Task 4 (I)- Desktop Calculator

This lab task involves creating a calculator application with a user-friendly interface using HTML, CSS, and FontAwesome for icons. Key features of the calculator include:

  • Header: A functional header with navigation icons for history and mode selection, providing easy access to calculator features.
  • Calculator Display: A large, constant display section that shows the current value, ensuring visibility during calculations.
  • Button Groups: Multiple button groups are arranged for different operations, including memory functions, arithmetic operators, and numeric input, offering a familiar calculator layout.
  • Color Scheme and Styling: Custom button styles are applied to distinguish between numeric, arithmetic, and functional buttons, while hover effects are used for enhanced interactivity.
  • External CSS: The external stylesheet (`style.css`) is used to define the overall layout, colors, and button styles, contributing to a clean and consistent visual appearance.
The layout is built using flexbox to ensure a responsive and neatly arranged design that works well across different screen sizes.

Lab Task 5 - Sports Week Website with Bootstrap 5

This page introduces the exciting activities planned for the COMSATS Islamabad Wah Campus Sports Week, along with information about the campus and a contact section. Key features include:

  • Sports Events: Engaging card layout showcasing the different sports events like "Blindfold Hunt," "Paper Plane Contest," and "Musical Chair." Each event card has a description and a "Register Now" button, making it easy for participants to sign up.
  • About Section: A detailed section providing information about COMSATS Islamabad Wah Campus. It includes an image and a description highlighting the campus's dedication to academic excellence, innovation, and student growth.
  • Contact Section: A contact form designed to facilitate communication regarding the Sports Week events. Users can provide their name, email, and message, ensuring their inquiries are addressed. The form uses external CSS for styling and maintains a responsive design for ease of use on different devices.
  • Footer: A simple footer with a reserved rights statement and a link to the official social media page of the campus.
Bootstrap 5 is used throughout the page for a consistent and responsive design, ensuring a pleasant user experience across various devices.

Lab Task 6 - Functional Calculator with JavaScript

This functional calculator is designed using HTML, External CSS, Bootstrap 5, and JavaScript, demonstrating key programming concepts. Key features include:

  • User Input: The calculator allows users to input numbers and operations using an intuitive interface, facilitating easy calculations.
  • Operations Supported: It supports basic arithmetic operations (addition, subtraction, multiplication, and division) as well as special functions like square, square root, and reciprocal.
  • Dynamic Display: The display updates in real-time, showing the current input and the ongoing calculation in a clear format, enhancing user interaction.
  • Responsive Design: Built with Bootstrap 5, the calculator is fully responsive, ensuring optimal performance on various devices.
This project showcases an understanding of JavaScript functions, event handling, and dynamic DOM manipulation, providing a practical tool for everyday calculations.
