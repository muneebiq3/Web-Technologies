Lab Task 1: Library Website

In this lab task, we created a basic library website using HTML and inline CSS. The website consists of multiple sections: a header with navigation links, an "About Us" section detailing the library's history, a catalog displaying featured books, a services section outlining the library's offerings, and a contact form. Inline CSS was used to style the layout and elements directly within the HTML file. The task focuses on structuring content while using basic styling techniques to enhance the user interface.

Lab Task 2 - edyou

Lab Task 2 involves the creation of a single-page website named "edyou" aimed at promoting a private social app for college life. The page includes several key features:

  • Header Section: Contains a menu button for navigation, a prominent main heading with the app's logo and tagline, and a call-to-action button for downloading the app from the Apple Store.
  • Main Content: Features an introductory banner with a description of the app and a modal window for users to join the waitlist.
  • Footer Section: Provides information about the app and additional navigation links.
The external CSS (`style.css`) handles styling, including background gradients, button styles, and responsiveness for various devices.
